## Overview
This is a **responsive dashboard layout** built with **HTML and CSS**. It includes a **sidebar**, **header**, **cards**, and **recent activities table**. This project demonstrates modern web design concepts such as **Flexbox**, **CSS Grid**, and **responsive design**.

---

## Features
- **Sidebar navigation** with hover effects
- **Header** with page title and profile icon
- **Cards section** displaying key metrics
- **Recent activities table** with status highlights
- Fully **responsive layout** for desktop and mobile screens
- Clean and professional **UI design** using shadows, borders, and spacing

---

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- 
## How to Run
1. Clone the repository or download the project files.
2. Open `index.html` in your browser.
3. Explore the sidebar, header, cards, and table layout.
